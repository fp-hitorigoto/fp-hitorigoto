# FP3級の勉強、「聴いて覚える」という選択肢。

「テキストを読んでも、なかなか頭に入らない…」

FP3級の勉強をしていて、そう感じたことはありませんか？

社会保険、年金、保険の種類、税金の控除…。覚えることが多くて、読んでも読んでも抜けていく感覚、ありますよね。

そんな方に向けて、FP3級の試験範囲を**楽曲で覚えられるシリーズ**をYouTubeで公開しました。

---

## 「歌で覚える」って、ちゃんと効果あるの？

疑わしく思う気持ち、わかります。でも考えてみると、小学校の頃に覚えた「九九」って、今も歌のリズムで出てきませんか？

あれと同じです。

メロディーと一緒に覚えた情報は、**言葉だけで覚えるより記憶に残りやすい**。これは記憶のしくみとして知られていることです。

「第1号・第2号・第3号被保険者って何だっけ」「定期保険と終身保険の違いは？」——そういう問いに、歌が浮かんで答えが出てくる感覚を目指して作りました。

---

## どんな内容が入っているの？

現在、Vol.1を公開しています。

**Vol.1 〜ライフプランニングと資金計画〜**（公開中）
公的年金の種別、老齢年金の受給要件、住宅ローンのしくみなど、FP3級の第1分野をカバーした10曲。

**Vol.2 〜リスク管理〜**（近日公開予定）
生命保険・損害保険・保険料のしくみなどをテーマに制作中です。チャンネル登録しておくと公開時に通知が届きます。

---

## いつ聴けばいいの？

通勤・通学の電車の中、家事をしながら、寝る前のリラックスタイム——**「ながら」で聴けるのが一番の強みです**。

テキストや問題集は「机に向かわないとできない」ですよね。でも音楽は違う。スマホのイヤホンさえあれば、いつでもどこでもインプットできます。

## おわりに

FP3級の勉強法は、テキスト＋問題集が王道です。でも、それだけが正解じゃない。

「聴いて覚える」「歌で思い出す」という方法が合う人も、きっといるはずです。

試験勉強が少しでも楽しくなれば、と思って作りました。よかったら聴いてみてくださいね。
